## Bold
>Use `** text **`

To make bold use **BOLD**  use 2 `**` astrick before and after text.

## Italic
> use `*text*` 

To make text to *Italic* use 1 `*` astrick before and after a text.

To use ~~Strick Though~~ use 2 `~~` tidle before and after a text.
